## Requirements
* iOS 8.0+
* Swift 2
* Xcode 7.0
 
## Installation

### Cocoapods
```ruby
use_frameworks!
pod "KCJogDial"
```
### Manually
To install manually the KCSelectionDialog in an app, just drag the `KCSelectionDialog/*.swift` file into your project.
